BBH Ecosystem

Gestionnaire de mots de passe et écosystème de sécurité utilisateur

Présentation

BBH Ecosystem est mon projet fil rouge : un gestionnaire de mots de passe combiné à un écosystème de sécurité pour les utilisateurs. Actuellement, les fonctionnalités incluent : surveillance VPN, suivi de connexion et gestion centralisée des accès. L’objectif est de créer un environnement sécurisé, intuitif et évolutif pour protéger les données sensibles.

Stack Technique

  • Node.js / REACT / MongoDB
  • REST API sécurisée
  • Gestion d’authentification et chiffrement
  • Monitoring VPN / Connexions utilisateur
  • TailwindCSS / DaisyUI pour l’interface
